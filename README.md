# Shoppy Shop

Vue shop application with Vuex & Firebase.

Deployed with Netlify.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
